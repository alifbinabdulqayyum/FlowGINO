# Flow Matching Geometry Informed Neural Operator

## Download Data & Install Conda Environment

Follow the data download procedure mentioned in [MMGN](https://github.com/Xihaier/Continuous-Field-Reconstruction-MMGN) paper. Also follow the conda environment installation procedure. Additionally, install [neuraloperator](https://neuraloperator.github.io/dev/index.html) package in the same conda environment.

## Train FlowGINO model

Run `run.sh` to train the model.

```
bash run.sh
```

## Test FlowGINO model

Run `test.sh` to test the trained models.

```
bash test.sh
```
